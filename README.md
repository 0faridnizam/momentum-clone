# Momentum Clone

A clone of the productivity chrome app [Momentum](https://chrome.google.com/webstore/detail/momentum/laookkfknpbbblfpciffpaejjkokdgca) on ReactJs

## Features

- [x] Clock
- [x] To Do List
- [x] Daily Photo
- [ ] Daily Quote
- [ ] Weather
- [ ] Offline Support
- [ ] local Storage
